# Evidencias verificables - Clean Marvel Album

Este directorio define el marco de verificacion tecnica y auditoria del TFM. La evidencia principal del caracter real del proyecto esta en el propio repositorio: historial de commits, workflows de GitHub Actions, suite de tests automatizados, arquitectura documentada y guias de despliegue. Esos artefactos son suficientes para evaluar el trabajo sin requerir material adicional.

Las capturas se consideran opcionales y solo aplican en auditorias externas o demos guiadas. Si se usan, se guardan en `docs/evidence/screenshots/` siguiendo una convencion de nombres con fecha. No son un requisito para la evaluacion del TFM.
